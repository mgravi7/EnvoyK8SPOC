## Usage

Open this file in:
- GitHub rendered view (https://github.com) to see shortcodes rendered
- Your editor's Markdown preview to test local rendering
- A browser opening a raw file to test system font fallback

If glyphs appear as question marks (?), install or configure fonts such as `Segoe UI Emoji`, `Segoe UI Symbol`, or Google Noto fonts and restart the viewer.
